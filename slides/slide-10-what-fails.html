Honest Warnings
✕

Shipping code you don't understand

AI can write it. You still need to debug it, maintain it, explain it.

✕

Trusting AI with facts

It hallucinates citations, invents endpoints, gets math wrong. Always verify.

✕

Pasting secrets into cloud AI

API keys, personal data, credentials. Use local models for sensitive work.

✕

Expecting perfection on first prompt

AI works through iteration. Start rough, give feedback, refine.

AI is a power tool. Power tools are great if you know what you're building.
