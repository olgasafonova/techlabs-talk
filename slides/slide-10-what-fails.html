Honest Warnings

What doesn't work

✕

Shipping code you don't understand

AI can write it. You still need to debug it, maintain it, and explain it to your team.

✕

Trusting AI with facts

It hallucinates citations, invents API endpoints, and gets math wrong. Always verify.

✕

Pasting secrets into cloud AI

API keys, personal data, credentials. Don't. Use local models (Ollama) for sensitive work.

✕

Expecting perfection on first prompt

AI works best through iteration. Start rough, give feedback, refine. Treat it like a conversation, not a search engine.

AI is a power tool. Power tools are great if you know what you're building.
